# golfdiagram
## Git

Detta projekt startade ursprungligen som något jag bara skulle sitta med hemma. Det primära syftet var från början bara att experimentera med Matplotlib och Seaborn för att lära mig mer inom detta område. 

Men desto mer jag jobbade med det insåg jag hur bra denna datan faktiskt var, och även att detta hade varit perfekt för kk1 uppgiften.

Eftersom arbetet har skett i en lokal fil på datorn, blev versionshanteringen i Git tyvärr inte som den skulle vara från start som kurskraven säger. Detta blir en viktig lärdom till kommande projekt att sätta upp ett Git repo i ett tidigare skede, även när det bara handlar om mindre arbeten. 

När projektet väl flyttades över till detta repo har fokus legat på att strukturera upp koden, dokumentera analysen samt hantera de sista optimeringarna och refaktoreringrna via löpande, beskrivande commits.
